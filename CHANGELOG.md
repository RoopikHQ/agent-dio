# Changelog

All notable changes to the Dio extension for Roopik IDE will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.37.2] - 2026-01-19

### Added
- Initial release of Dio for Roopik IDE
- Roopik Designer Mode for visual development and project management
- Marketplace auto-update support
- Roopik Canvas integration tools
- Component management system (add, edit, remove)
- Multi-mode AI assistance (Code, Architect, Ask, Debug, Designer, Custom)
- Support for multiple AI providers (OpenAI, Anthropic, Google, OpenRouter)
- Secure API key storage
- Context-aware code generation
- Multi-language support

### Changed
- Rebranded from Roo Code to Dio for Roopik IDE
- Updated documentation and branding
- Configured for Roopik-specific features

### Fixed
- Canvas component batch operations (now sequential to prevent race conditions)
- Type definitions for global state keys
- Menu icon visibility in auxiliary sidebar
